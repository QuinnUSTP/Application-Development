## 📞 Support & Troubleshooting

### **If products don't show:**
1. Check browser console (F12)
2. Verify data/products.json exists
3. Check image paths
4. Clear browser cache (Ctrl+Shift+Delete)
5. Refresh page (Ctrl+F5)

### **If add to cart doesn't work:**
1. Make sure JavaScript is enabled
2. Check browser console for errors
3. Try different browser
4. Verify localStorage is enabled

### **If page looks broken:**
1. Check style.css is loading
2. Verify all CSS classes are present
3. Check for browser compatibility (use Chrome/Firefox)

## 📝 Files You Can Modify

### **Easy Customizations:**

1. **Change Colors** - Edit CSS variables in style.css:
   ```css
   :root {
       --highlight-color: #ff523b; /* Change this */
       --dark-highlight-color: #563434;
   }
   ```

2. **Add More Products** - Add to data/products.json:
   ```json
   {
     "id": 13,
     "name": "New Product",
     "price": 99.99,
     "image": "images/product-13.jpg",
     "rating": 4,
     "category": "clothing"
   }
   ```

3. **Update Testimonials** - Edit HTML in index.html (lines 110-170)

4. **Change Product Counts** - Edit limits in js/index.js:
   ```javascript
   const response = await apiService.getProducts({ limit: 4 }); // Change 4
   ```
