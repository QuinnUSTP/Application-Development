## 📱 Responsive Breakpoints

### Desktop (1300px max-width)
```
- Full layout
- 4-column product grid
- 3-column testimonials
- 5-column brands
- Menu fully visible
```

### Tablet (≤800px)
```
- Navigation menu collapses
- Hamburger menu icon appears
- Menu dropdown on click
- Grid layouts maintained
```

### Mobile (≤600px)
```
- Single column layout
- All grids stack vertically
- Full-width products
- Touch-friendly buttons
- Collapsible menu
- Simplified footer
```

## 🎬 Animations & Effects

```
Product Cards:
- Hover: translateY(-5px) - moves up slightly
- Transition: 0.5s ease
- Shadow: 0 0 20px rgba(0,0,0,0.1)

Testimonial Cards:
- Hover: translateY(-10px) - bigger lift
- Box shadow on hover
- Smooth transition

Buttons:
- Hover: Background color change
- Active: Scale(0.98) - press effect
- Transition: all 0.3s

Logo Brands:
- Default: Grayscale(100%)
- Hover: Grayscale(0%)
- Smooth transition
```
