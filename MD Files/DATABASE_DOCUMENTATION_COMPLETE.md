## 🎯 THE 6 KEY CHECKS

Every guide teaches you these 6 essential checks:

```
1. Is MongoDB Running?
   Command: Get-NetTCPConnection -LocalPort 27017
   Expected: Connection on 127.0.0.1:27017

2. Is Database Seeded?
   Command: db.products.countDocuments()
   Expected: Returns 12

3. Is Backend Connected?
   Look for: "✅ MongoDB Connected: localhost"
   Expected: In backend console

4. Is Frontend Working?
   Look for: "✅ Products fetched from backend: 12"
   Expected: In browser console (F12)

5. Can Users Register?
   Action: Register a user
   Check: db.users.countDocuments() increases
   Expected: Increases by 1

6. Can Orders Save?
   Action: Checkout an order
   Check: db.orders.countDocuments() increases
   Expected: Increases by 1

ALL 6 ✅ = System is PERFECT!
```
